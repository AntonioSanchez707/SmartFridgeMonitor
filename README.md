# SmartFridgeMonitor

Proyecto desarrollado por SaiviLab.

## Objetivo
Construir un monitor inteligente para refrigeradores utilizando un ESP32 capaz de medir:

- Temperatura del freezer.
- Temperatura de conservación.
- Corriente del compresor mediante SCT-013-000.
- Potencia y consumo energético.
- Ciclos de trabajo del compresor.
- Dashboard web en tiempo real mediante WiFi.

## Hardware
- ESP32 DevKit (38 pines)
- 2 × SB-41 Full Gauge (NTC 10 kΩ)
- SCT-013-000
