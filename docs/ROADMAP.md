# SmartFridgeMonitor - Roadmap

## Objetivo
Desarrollar un monitor inteligente para refrigeradores basado en ESP32 con interfaz web.

## Fase 1
- [x] Conectividad WiFi
- [x] Servidor web básico
- [ ] Lectura de NTC Freezer
- [ ] Lectura de NTC Conservación
- [ ] Lectura SCT-013-000

## Fase 2
- [ ] API JSON
- [ ] Dashboard dinámico
- [ ] Gráficas en tiempo real

## Fase 3
- [ ] Registro histórico
- [ ] Diagnóstico automático
- [ ] Calibración

## Hardware
- ESP32 DevKit 38 pines
- 2x SB-41 Full Gauge (NTC 10k)
- SCT-013-000
